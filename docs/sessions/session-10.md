[감염관리 기본 지침]

■ 손 위생
- 손 위생은 감염 예방의 가장 중요한 방법이다.
- 손 위생은 환자 접촉 전후, 체액 노출 후 반드시 시행한다.
- 손 위생 방법: 물과 비누 또는 알코올 손 소독제 사용
- 손 위생 시간: 최소 20초 이상

■ 개인 보호구(PPE)
- 개인 보호구는 상황에 맞게 착용한다.
- 기본 보호구: 마스크, 장갑, 가운, 고글/안면보호대
- 착용 순서: 가운 → 마스크 → 고글 → 장갑
- 탈의 순서: 장갑 → 고글 → 가운 → 마스크
- 사용 후 보호구는 지정된 장소에 폐기한다.

■ 환경 관리
- 환자 접촉 물품은 사용 후 소독한다.
- 고빈도 접촉 표면은 매일 소독한다.
- 의료 폐기물은 지침에 따라 분리 배출한다.

■ 감염 의심 환자 관리
- 감염 의심 환자는 별도 관리한다.
- 의심 증상: 발열, 호흡기 증상, 설사 등
- 즉시 담당자에게 보고하고 격리 여부를 결정한다.

🔹 예제데이터 ① 감염관리 기본 지침 (기본)
형식: 텍스트 (수강생 배포용)
[감염관리 기본 지침]

■ 손 위생
- 손 위생은 감염 예방의 가장 중요한 방법이다.
- 손 위생은 환자 접촉 전후, 체액 노출 후 반드시 시행한다.
- 손 위생 방법: 물과 비누 또는 알코올 손 소독제 사용
- 손 위생 시간: 최소 20초 이상

■ 개인 보호구(PPE)
- 개인 보호구는 상황에 맞게 착용한다.
- 기본 보호구: 마스크, 장갑, 가운, 고글/안면보호대
- 착용 순서: 가운 → 마스크 → 고글 → 장갑
- 탈의 순서: 장갑 → 고글 → 가운 → 마스크
- 사용 후 보호구는 지정된 장소에 폐기한다.

■ 환경 관리
- 환자 접촉 물품은 사용 후 소독한다.
- 고빈도 접촉 표면은 매일 소독한다.
- 의료 폐기물은 지침에 따라 분리 배출한다.

■ 감염 의심 환자 관리
- 감염 의심 환자는 별도 관리한다.
- 의심 증상: 발열, 호흡기 증상, 설사 등
- 즉시 담당자에게 보고하고 격리 여부를 결정한다.


🔹 예제데이터 ② 손 위생 상세 지침
형식: 텍스트 (확장용)
[손 위생 상세 지침]

■ 손 위생이 필요한 5가지 순간 (WHO 기준)
1. 환자 접촉 전
2. 무균/청결 시술 전
3. 체액 노출 위험 후
4. 환자 접촉 후
5. 환자 주변 환경 접촉 후

■ 손 씻기 방법 (물과 비누)
1. 손을 물에 적신다
2. 비누를 충분히 묻힌다
3. 손바닥을 마주 대고 문지른다
4. 손등을 문지른다
5. 손가락 사이를 문지른다
6. 손톱 밑을 문지른다
7. 엄지를 돌려가며 문지른다
8. 손목을 문지른다
9. 흐르는 물에 헹군다
10. 일회용 타월로 물기를 제거한다

■ 손 소독제 사용 방법 (알코올 기반)
1. 손 소독제를 손바닥에 덜어낸다 (3ml 정도)
2. 손 전체에 골고루 문지른다
3. 완전히 마를 때까지 문지른다 (약 20~30초)

■ 손 위생 시 주의사항
- 손에 눈에 보이는 오염이 있으면 반드시 물과 비누 사용
- 알코올 손 소독제는 눈에 보이는 오염이 없을 때 사용
- 손톱은 짧게 유지
- 반지, 시계 등 장신구 제거 권장

■ 접촉주의 격리
- 1인실 격리 또는 코호트 격리

- 배양검사 결과 등 확인 후 해제

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 감염관리는 왜 중요한가요?
감염관리는 환자와 의료진 모두를 보호하는 
기본적인 안전 활동입니다. 
작은 실천이 큰 감염을 예방합니다.

■ 손 위생
- 환자를 만나기 전과 후에 반드시 손을 씻으세요.
- 체액에 노출되면 즉시 손을 씻으세요.
- 비누와 물 또는 알코올 손 소독제를 사용하세요.
- 최소 20초 이상 꼼꼼히 씻어야 합니다.

■ 개인 보호구(PPE) 착용
- 상황에 맞는 보호구를 선택해서 착용하세요.
- 기본 보호구: 마스크, 장갑, 가운, 고글
- 착용 순서: 가운 → 마스크 → 고글 → 장갑
- 벗는 순서: 장갑 → 고글 → 가운 → 마스크
- 사용 후에는 지정된 장소에 올바르게 폐기하세요.

■ 환경 관리
- 환자가 사용한 물품은 사용 후 소독하세요.
- 자주 만지는 곳(손잡이, 난간 등)은 매일 소독합니다.
- 의료 폐기물은 분리해서 버려주세요.

■ 감염 의심 환자 발견 시
- 발열, 기침, 설사 등의 증상이 있으면 주의하세요.
- 의심 환자는 즉시 담당자에게 보고해주세요.
- 격리 여부는 담당자가 결정합니다.
