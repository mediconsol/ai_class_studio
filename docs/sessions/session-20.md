[슬라이드 10. 이 과정에서 배운 것]
화면 문구
■ 이 과정에서 배운 것

1. AI는 도구다
   - 판단은 사람, 정리는 AI

2. 질문이 아니라 지시다
   - 프롬프트 = 업무 지시서

3. 자동화는 생각의 전환이다
   - 패턴 인식 → 구조화 → 반복 제거

4. 기준이 있으면 안전하다
   - 개인정보, 판단, 책임

5. 작게 시작해서 넓힌다
   - 나 → 동료 → 부서 → 병원

강사 멘트
이 과정에서 배운 것을 정리합니다.
AI는 도구입니다. 판단은 사람, 정리는 AI. 질문이 아니라 지시입니다. 프롬프트는 업무 지시서. 자동화는 생각의 전환입니다. 패턴 인식 → 구조화. 기준이 있으면 안전합니다. 개인정보, 판단, 책임. 작게 시작해서 넓힙니다. 나 → 동료 → 부서.
