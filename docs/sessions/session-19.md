🔷 개념 이해 - 확산 전략 (8분)

[슬라이드 10. 확산은 '한 번에' 하면 실패한다]
화면 문구
■ 확산의 현실

❌ 잘못된 접근
"이거 다들 쓰세요" → 아무도 안 씀

⭕ 올바른 접근
"나 → 옆 동료 → 부서 → 병원"

→ 작게 시작, 단계적 확산

강사 멘트
확산에 대한 현실입니다.
"이거 다들 쓰세요" → 아무도 안 씁니다.
확산은 단계적으로 해야 합니다. 나 → 옆 동료 → 부서 → 병원
작게 시작해서 점점 넓히는 게 성공 확률이 높습니다.

[슬라이드 11. 확산 3단계 모델]
화면 문구
■ 확산 3단계 모델

┌─────────────────────────────────────┐
│  1단계: 개인 사용                   │
│  - 나 혼자 써보기                   │
│  - 문제점 파악                      │
├─────────────────────────────────────┤
│  2단계: 부서 공유                   │
│  - 같은 근무조, 같은 부서           │
│  - "이거 써보세요" 권유             │
├─────────────────────────────────────┤
│  3단계: 병원 확산                   │
│  - 공식 채택                        │
│  - 표준화                           │
└─────────────────────────────────────┘

→ 이번 과정에서는 1~2단계까지가 목표

강사 멘트
확산 3단계 모델입니다.
1단계: 개인 사용. 나 혼자 써보기. 2단계: 부서 공유. 같은 근무조, 같은 부서에 권유. 3단계: 병원 확산. 공식 채택, 표준화.
이번 과정에서는 1~2단계까지가 목표입니다. 3단계는 병원 차원의 결정이 필요합니다.

[슬라이드 12. 단계별 확산 방법]
화면 문구
■ 단계별 확산 방법

[1단계: 개인 사용]
- 1주일 정도 직접 사용
- 불편한 점 기록
- 수정 필요한 부분 파악

[2단계: 부서 공유]
- 1~2명에게 먼저 공유
- 사용 방법 간단히 설명
- 피드백 수집
- 필요시 수정

[3단계: 병원 확산]
- 관리자 보고
- 공식 승인
- 교육 자료화

강사 멘트
단계별 확산 방법입니다.
1단계: 1주일 정도 직접 사용하면서 불편한 점 파악. 2단계: 1~2명에게 먼저 공유하고 피드백 수집. 3단계: 관리자 보고, 공식 승인, 교육 자료화.
오늘은 1~2단계 계획을 세웁니다.

[슬라이드 13. 확산 시 흔한 실수]
화면 문구
■ 확산 시 흔한 실수

1. 너무 빨리 확산
   - 검증 없이 바로 전파 → 신뢰 저하

2. 강제 사용
   - "다 쓰세요" → 반발

3. 설명 없이 공유
   - 파일만 던지기 → 안 씀

4. 피드백 무시
   - "내가 만들었으니까 맞아" → 개선 안 됨

5. 유지보수 안 함
   - 한 번 만들고 끝 → 곧 낡음

강사 멘트
확산 시 흔한 실수입니다.
너무 빨리 확산: 검증 없이 바로 전파하면 신뢰가 떨어집니다. 강제 사용: "다 쓰세요"는 반발을 삽니다. 설명 없이 공유: 파일만 던지면 안 씁니다. 피드백 무시: 개선이 안 됩니다. 유지보수 안 함: 한 번 만들고 끝내면 곧 낡습니다.

[슬라이드 14. 성공적인 확산의 조건]
화면 문구
■ 성공적인 확산의 조건

1. 먼저 본인이 충분히 사용
   - "나도 쓰고 있어요"

2. 작게 시작
   - 1~2명부터

3. 사용법 설명 포함
   - "이렇게 쓰면 돼요"

4. 피드백 반영
   - 수정하면서 개선

5. 강요하지 않음
   - "해보실래요?" (권유)

강사 멘트
성공적인 확산의 조건입니다.
먼저 본인이 충분히 사용. "나도 쓰고 있어요" 작게 시작. 1~2명부터. 사용법 설명 포함. "이렇게 쓰면 돼요" 피드백 반영. 수정하면서 개선. 강요하지 않음. "해보실래요?"로 권유.
