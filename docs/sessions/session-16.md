🔷 개념 이해 (7분)

[슬라이드 5. 좋은 과제의 조건]
화면 문구
■ 좋은 과제의 5가지 조건

1. 실제 업무다
   - 가상이 아닌 내 실제 업무

2. 혼자 해볼 수 있다
   - 다른 부서 협조 불필요

3. 결과물이 남는다
   - 문서/템플릿/양식 등

4. 바로 적용할 수 있다
   - 교육 끝나고 바로 사용

5. 범위가 명확하다
   - "이것까지만" 선이 있음

강사 멘트
좋은 과제의 조건입니다.
실제 업무입니다. 가상이 아니라 진짜 하는 일. 혼자 해볼 수 있습니다. 다른 부서 협조 불필요. 결과물이 남습니다. 문서, 템플릿, 양식 같은 것. 바로 적용할 수 있습니다. 교육 끝나고 바로 사용. 범위가 명확합니다. "여기까지만"이라는 선이 있음.
이 다섯 가지를 충족하면 좋은 과제입니다.

[슬라이드 6. 과제로 적합한 업무 예시]
화면 문구
■ 과제로 적합한 업무 예시

• 간호기록 문장 정리 자동화
• 인수인계 요약 템플릿 제작

• 적정성평가 대응 문서 구조화
• 감염관리 안내문 다중 포맷 제작
• SOP → 체크리스트 변환

강사 멘트
과제로 적합한 업무 예시입니다.
6차시부터 14차시까지 우리가 했던 것들이 그대로 과제가 될 수 있습니다.
15차시에서 시나리오를 만드셨다면 그걸 그대로 과제로 발전시키면 됩니다.

[슬라이드 7. 과제로 부적합한 경우]
화면 문구
■ 과제로 부적합한 경우

❌ 너무 크다
   - "병원 전체 업무 개선"

❌ 여러 부서 협의 필요
   - "타 부서와 협업 시스템"

❌ 시스템 구축 전제
   - "새로운 프로그램 도입"

❌ 결과물이 불명확
   - "AI 활용 능력 향상"

❌ 판단/결정 업무
   - "진단 보조", "처방 추천"

강사 멘트
과제로 부적합한 경우입니다.
너무 크면 안 됩니다. 다른 부서 협의가 필요하면 안 됩니다. 시스템 구축이 전제되면 안 됩니다. 결과물이 불명확하면 안 됩니다. 판단/결정 업무는 절대 안 됩니다.
이런 경우 범위를 줄여야 합니다.

[슬라이드 8. 개인 과제 vs 부서 과제]
화면 문구
■ 개인 과제 vs 부서 과제

개인 과제
- 혼자 개선
- 내 업무 범위 내
- 책임 주체: 본인

부서 과제 (선택)
- 같은 부서 동료와 함께
- 소규모 공유 (2~3명)
- 책임 주체: 명확히 지정

→ 이번 과정에서는 개인 과제 권장

강사 멘트
개인 과제와 부서 과제가 있습니다.
개인 과제는 혼자 하는 것. 부서 과제는 동료와 함께 하는 것.
부서 과제도 가능하지만, 이번 과정에서는 개인 과제를 권장합니다.
혼자 해야 책임과 범위가 명확해집니다.

[슬라이드 9. 과제 정의서란?]
화면 문구
■ 과제 정의서 = 과제의 설계도

"무엇을, 왜, 어떻게, 어디까지"

→ 과제 정의서가 잘 되어 있으면
   17~18차시가 쉬워진다

→ 과제 정의서가 모호하면
   뒤에서 계속 헤맨다

강사 멘트
과제 정의서는 과제의 설계도입니다.
무엇을, 왜, 어떻게, 어디까지 할 것인지 명확하게 적는 문서입니다.
이게 잘 되어 있으면 17~18차시가 쉬워집니다. 이게 모호하면 뒤에서 계속 헤맵니다.
오늘 시간을 충분히 들일 가치가 있습니다.

■ 검토 결과

1. 범위 적절성: ⭕ 적절
   - "환자 5명"으로 숫자 명시
   - 혼자서 충분히 할 수 있는 범위
   - 전체 병동이 아닌 샘플로 시작

2. 결과물 명확성: ⭕ 명확
   - "요약 템플릿"이라는 구체적 결과물
   - AI 프롬프트 + 결과물 예시

3. 실현 가능성: ⭕ 높음
   - 7차시에서 유사 실습 경험
   - 데이터(간호기록) 이미 있음
   - 개인정보 처리 방안만 확인 필요

4. 더 구체적으로 만들 수 있는 부분
   - 요약 형식을 미리 정해두면 좋음
     (예: 현재상태 1줄 / 주의사항 1줄 / 확인사항 1줄)
   - 입력 데이터 형태 명시
     (EMR 복사? 메모 형태?)

5. 예상 어려움
   - 간호기록 시스템에서 복사하는 과정
   - 개인정보(환자명) 처리 방안
   - 병원 인수인계 양식과의 일치 여부

■ 좋은 점
- 명확한 범위 설정
- 구체적인 기대 효과 (20분→10분)
- 실제 업무와 직결

■ 대안 3가지

1. 민원 응대 표준 문구집 제작
   - 가장 직관적
   - 결과물이 명확 (문구집)

   - 범위(원무) 명시

   - 숫자(5대) 포함으로 범위 명확
   - 가장 구체적
