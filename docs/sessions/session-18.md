[슬라이드 4. 왜 결과물이 중요한가]
화면 문구
■ 왜 결과물이 중요한가

1. 교육 효과의 증거
   - "배웠다" vs "만들었다"

2. 현업 적용의 시작점
   - 결과물 없으면 적용도 없음

3. 다른 사람에게 공유 가능
   - "해봤는데 좋아요" vs "이거 써보세요"

4. 본인의 자산
   - 교육 끝나도 남는 것

강사 멘트
결과물이 중요한 이유입니다.
교육 효과의 증거입니다. "배웠다"보다 "만들었다"가 훨씬 강력합니다.
현업 적용의 시작점입니다. 결과물 없으면 적용도 시작이 안 됩니다.
공유가 가능합니다. "해봤는데 좋아요"보다 "이거 써보세요"가 설득력 있습니다.
본인의 자산입니다. 교육 끝나도 남는 게 있어야 합니다.

[슬라이드 6. 좋은 결과물의 특징]
화면 문구
■ 좋은 결과물의 5가지 특징

1. 설명 없이도 쓸 수 있다
   - 보는 순간 쓰임새가 보임

2. 한 장(또는 한 세트)으로 끝난다
   - 길면 안 씀

3. 다시 만들 필요가 없다
   - 반복 사용 가능

4. AI와 함께 쓸 수 있다
   - 프롬프트가 포함되어 있음

5. 수정이 쉽다
   - 상황에 맞게 조정 가능

강사 멘트
좋은 결과물의 특징입니다.
설명 없이도 쓸 수 있습니다. 보면 바로 알 수 있어야 합니다. 한 장으로 끝납니다. 길면 안 씁니다. 다시 만들 필요 없습니다. 반복 사용 가능. AI와 함께 쓸 수 있습니다. 프롬프트 포함. 수정이 쉽습니다. 상황에 맞게 조정 가능.
이 다섯 가지를 기억하세요.
